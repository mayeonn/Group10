🎧Music Playlist
=================
group10의 미니 프로젝트 주제는 음악 플레이리스트 프로그램입니다.   
## 기능
   - CRUD
      - 플레이리스트에 곡을 추가, 정보 수정, 삭제
      - 현 플레이리스트 조회
   - 파일
      - 현재 음악 리스트를 파일에 저장
      - 다음 실행시 자동으로 저장된 플레이리스트가 조회됨
   - 검색
      -  곡 제목, 가수, 앨범명으로 검색
   - 정렬
      - 곡 제목에 따라 오름차순 정렬
      - 좋아요 수 많은 순대로 정렬
